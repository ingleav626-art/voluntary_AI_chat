## P0：基础框架搭建

**状态：已完成**

- Maven 多模块项目结构（common + ai-core + server + client）
- Spring Boot 3.5.15 + JavaFX 17 + MyBatis-Plus 3.5.15
- WebSocket 通信框架（Spring WebSocket Handler + Java WebSocket 客户端）
- REST API 基础框架（Spring Security + JWT 认证过滤器）
- 数据库连接（MySQL + H2 双模式）
- Flyway 数据库迁移
- 代码质量工具链（Checkstyle + SpotBugs + JaCoCo）

---

## P1：用户系统 + 基础聊天

**状态：已完成**

### 用户系统
- 手机号注册（短信验证码）
- JWT 登录认证（Access Token 2h + Refresh Token 7d）
- 个人资料管理（头像、昵称、签名、性别、年龄、生日、详细说明）
- 密码修改、手机号换绑、忘记密码
- 用户搜索

### 基础聊天
- 单聊文字消息（WebSocket 实时推送 + REST 备用通道）
- 消息持久化存储（MySQL/H2）
- 消息已读回执
- 历史消息分页加载
- 会话列表

---

## P2：群聊 + 好友系统

**状态：已完成**

### 好友系统
- 好友申请 → 同意/拒绝流程
- 好友列表、好友备注、删除好友

### 群聊
- 创建群组、邀请/移除成员、退出群组
- 群主转让、解散群组、设置/取消管理员
- 群昵称、群头像、群公告（支持置顶）
- 群消息广播
- 消息撤回（2 分钟内，支持管理员撤回群成员消息）
- 图片消息（上传、压缩、缩略图、本地文件存储）

---

## P3：AI 功能集成

**状态：进行中**

### 已完成
- AI 角色 CRUD（名称、人设、系统提示词、模型提供商、加密 API Key）
- AI 单聊（WebSocket 流式输出，支持 OpenAI/DeepSeek/通义/智谱）
- API Key AES-256-GCM 加密存储
- AI 角色管理界面（创建、编辑、删除）
- ai-core 模块拆分（独立 Maven 模块，支持 POJO 和 Spring 双模式）
- LocalAiEngine 本地 AI 引擎（纯 POJO，H2 JDBC 直连）
- 三包分发架构（客户端包/测试包/云端包）
- 多 AI 群配置接口（创建、查询群 AI 配置）
- AI 流式回复 WebSocket 推送（AI_STREAM 消息类型）

### 进行中
- 多 AI 群触发逻辑（@提及、关键词触发、概率触发的完整实现）
- AI 记忆管理（记忆摘要生成、关键词提取、重要度评分）

### 计划中
- AI 主动聊天（定时任务，随机 5-30 分钟间隔触发）
- 记忆 RAG 检索（当前用关键词匹配，后续接入向量数据库）

## 风险管理

### 技术风险
- WebSocket 稳定性：已实现断线重连机制（RECONNECT/RECONNECT_ACK）
- AI 接口限流：依赖 LLM 提供商限流策略
- 本地模型性能：规划中，待评估 DJL + ONNX 方案

### 协作风险
- 接口不一致：先定义接口再开发（已通过 API.md 规范）
- 代码冲突：feature/A/client 和 feature/B/server 分支隔离
- 进度不同步：每日站会同步进度
