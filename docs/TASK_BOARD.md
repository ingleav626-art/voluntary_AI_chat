# 任务看板

> 两人并行开发的任务分配和进度跟踪

---

## 使用说明

- **开发者A**：负责客户端（JavaFX）
- **开发者B**：负责服务端（Spring Boot）
- 每日站会同步进度
- 完成的任务移动到"已完成"列
